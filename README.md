## How It Works

1. Customer searches for destination, dates, and travelers
2. Platform returns aggregated results from multiple inventory sources
3. Customer selects flights, hotels, or packages
4. Dynamic packaging creates a bundled itinerary
5. Payment processed with secure travel payment gateway
6. Voucher and itinerary sent to customer; merchant receives booking
